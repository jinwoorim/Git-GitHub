# Git 설치

- [https://git-scm.com/](https://git-scm.com/) 로 이동해서 Git 다운

- 버전확인

```shell
git --version
```
- 협업시 윈도우와 맥에서 엔터 방식 차이로 인한 오류 방지

```shell
git config --global core.autocrlf true
```
